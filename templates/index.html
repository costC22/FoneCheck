Como usar o FoneCheck

1

Selecione o Tipo

Escolha entre Burger King ou Popeyes para definir o tipo de busca.

2

Digite o Código

Insira o código da loja (BK Number ou PLK Number) no campo de busca.

3

Busque os Telefones

Clique em "Buscar Telefones" para encontrar os números associados ao código.

4

Exporte os Dados

Use os botões de exportação para salvar os resultados em arquivo Excel.

Dica Pro: Use os filtros para encontrar telefones específicos ou excluir números vazios dos resultados.
