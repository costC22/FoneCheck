Como usar

  1. Selecione o tipo de busca: OJA (códigos 15xxx, 18xxx, 20xxx, 22xxx, 30xxx) ou Popeyes (PLK + códigos 12xxx, 13xxx, 14xxx)
  2. Digite o código do colaborador que deseja buscar
  3. Clique em "Buscar" para encontrar os telefones
  4. Use os filtros para refinar os resultados
  5. Exporte os resultados se necessário
